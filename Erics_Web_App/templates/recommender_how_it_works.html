How it works

This page tells you a bit about how Meeple Finder Mediocre Board Game Recommender works, what techniques are used, and where the data comes from!

The model I use to recommend games relies entirely on data collected from boardgamegeek.com. First the user rating data is used as input in a Spark ALS model to create two matrices that can be used to predict the ratings that a particular user might rate a game.

The model takes user data in the form of three games that the user 'likes a lot'. This input is then used to find the three users in the database that are most similar to this user using a cosine similarity measure. The model then calls then collects and averages of the predicted ratings for all of the games in the database for these three users (calculated previously using ALS's transform method).

These ratings are then averaged together with the average user rating of the games. Finally, the top three games are taken from this list and returned to the user as their recommendations.
